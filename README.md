## ¿Cómo obtener el link RAW de un archivo .xlsx?

Para obtener el link RAW de un archivo `.xlsx` (u otro archivo binario) en GitHub:

1. Ve al repositorio en GitHub.
2. Haz clic en el archivo que deseas descargar (por ejemplo, `estudiantes.xlsx`).
3. Haz clic en el botón **"Raw"** o **"Download"** en la parte superior derecha del visor de archivos.
4. Copia la URL del navegador. Tendrá el formato:

```
https://raw.githubusercontent.com/{usuario}/{repositorio}/{rama}/{nombre-del-archivo}
```

**Ejemplo:**
```
https://raw.githubusercontent.com/Loldeluis/Formatos/main/estudiantes.xlsx
```

> **Nota:** Para archivos con espacios en el nombre, los espacios se reemplazan por `%20` en la URL. Por ejemplo, `estudiantes (2).xlsx` se convierte en:
> ```
> https://raw.githubusercontent.com/Loldeluis/Formatos/main/estudiantes%20(2).xlsx
> ```
